Sentinel Plugin

Minecraft Anti-Xray Detection Plugin

Overview

Sentinel is a Minecraft server plugin that detects players using X-ray by analyzing mining behavior and identifying suspicious ore acquisition patterns. It helps server admins maintain fair gameplay with lightweight, practical checks. The plugin can emit structured logs suitable for SIEM ingestion (e.g., Wazuh) to centralize alerts and investigations.

Highlights

  • Monitors ore mining events to detect abnormal patterns
  • Flags high-likelihood X-ray behavior for admin review
  • Lightweight and server-friendly design

Tech Stack

Java Spigot/Bukkit Minecraft Anti-Cheat
