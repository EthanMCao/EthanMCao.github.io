How it's built

One piece,
four stages.

The same canyon composition tracked from a value thumbnail to a rendered frame. Values come first, then local colour, then light and air — never the other way around.

How I paint

Values first. If a piece doesn't read in greyscale, no amount of colour will save it. I lock the value structure and the focal point before anything else.

Big to small. I block the largest shapes, then the next size down, and only render detail where the eye actually lands. The rest stays soft on purpose.

Light logic. One committed light source, consistent everywhere — warm key, cooler shadows, a rim to lift the subject off the background, and air thickening into the distance.
